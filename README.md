## 概述

之所以做这个版本主要是因为原本的 pangu.js 对内容做了过度的转化，有的东西不是我想要的，我只是想要加个空格。

所以特别做了这个版本来处理排版。

## 修改

- 去除了对中间点的转换
- 去除了标点强制到Full-width的转换
- 替换原有正则为一套更加适合普通写作的新正则
- 重写测试代码
